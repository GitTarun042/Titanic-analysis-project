# Titanic Survival Analysis

## Problem Statement

Analyze Titanic passenger data to identify factors that influenced survival during the disaster. The project focuses on passenger demographics, ticket class, age, and gender to uncover survival patterns.

## Objectives

* Clean and preprocess Titanic data.
* Handle missing values.
* Analyze survival rates across different passenger groups.
* Visualize survival patterns using charts.
* Generate meaningful insights from the dataset.

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Google Colab

## Analysis Performed

* Missing value treatment
* Age and fare imputation
* Survival rate by gender
* Survival rate by passenger class
* Survival rate by age group
* Data visualization and insight generation

## Key Insights

* Female passengers had significantly higher survival rates than males.
* First-class passengers were more likely to survive.
* Survival patterns varied across age groups.
* Passenger demographics strongly influenced survival outcomes.
